# 🎨 Curated AI Agent Prompt Library for MLX Diffusion

A curated library of high-performance prompt templates optimized for **FLUX.2 Klein (4B/9B)**, **Sana 2.0 Sprint**, and **Wan 2.1** diffusion models. AI Agents can use these structured prompts for automated asset generation.

---

## 1. 📷 Studio Photorealism

### Template Formula:
`[Subject], [Lighting], 8k RAW photo, sharp focus, cinematic composition, depth of field, masterpiece`

### Example Prompts:
- **Portrait**: `Cinematic studio portrait of a woman in golden hour sunlight, 8k RAW photo, sharp focus, shallow depth of field, masterpiece`
- **Product Photography**: `Minimalist ceramic coffee mug on a dark marble table, soft studio side-lighting, 8k RAW photo, ultra detailed texture`

## 2. 💎 UI/UX & Glassmorphic 3D Renders
